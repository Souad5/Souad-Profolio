# 💼 Souad Al Kabir – Developer Portfolio

Welcome to my personal portfolio website built using React, Tailwind CSS, DaisyUI, and Vite. This site showcases my skills, projects, education, and contact information.

## 🚀 Features

- Responsive navigation bar
- Introduction and professional designation
- Resume download button
- Social links (GitHub, LinkedIn)
- About me section with personality and hobbies
- Skills section with animated bars
- Education background
- Projects section with live demo and GitHub links
- Contact form powered by EmailJS
- Footer section
- Mobile-friendly and clean UI with TailwindCSS + DaisyUI
- Animations with Framer Motion

## 🛠️ Tech Stack

- **React** 19+
- **Vite**
- **Tailwind CSS** 4
- **DaisyUI**
- **React Router** v7
- **Framer Motion**
- **EmailJS**
